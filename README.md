## **Using 📦**

1. Clone Template

```
git clone https://github.com/yurineves1994/model-react-ts
```

2. Install Packages

```
pnpm install
```

3. Start Project

```
pnpm dev
```

4. If you using git, delete the existing folder .git after cloning (open `git bash` or other terminal)

```
rm -rf .git
```

## **Options ✍️**

1. Check lint

```
pnpm lint
```

2. Fix lint

```
pnpm lint:fix
```

3. Check prettier

```
pnpm prettier
```

4. Fix prettier

```
pnpm prettier:fix
```

5. Fix lint and prettier

```
pnpm format
```
